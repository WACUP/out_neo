This directory contains audio parsers (descendants of Parser class) 
and support classes

* AC3Parser (ac3\ac3_parser.h): AC3 parser
* DTSParser (dts\dts_parser.h): LibDTS based parser
* MPAParser (mpa\mpa_parser.h): Valex MPEG1/2 Layer I/II audio parser

* FileParser (file_parser.h): adapter class to work with compressed files.
